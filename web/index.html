About Workflow Tracker

Repository Created January 30, 2026
Created By David Citarelli
AI-Assisted Development OpenAI Codex (GPT-5)

What This Program Is Supposed To Do

  • Track onboarding progress in a local-first Kanban workflow.
  • Manage candidate records, weekly notes, and todos in one place.
  • Protect data using password-based encryption by default.

What This Program Is Not Supposed To Do

  • Make legal, HR, or compliance decisions for your team.
  • Automatically upload your data to external servers by default.
  • Replace your backup and data-governance responsibilities.

How People Can Contribute

  1. Search existing issues first, then open a focused new issue if needed.
  2. Include version, platform, repro steps, and expected vs. actual results.
  3. Fork, branch, implement, run lint/tests, and open a pull request.

    Read This First

    You have selected to import the database .

    If you select ADD / APPEND, we will import everything from this database and append it to your current database, only if every column and data type matches. We will run a compatibility and security scan first.

    If you select REPLACE, we will overwrite your current working database with the one you just selected. If you select VIEW, we will import it into the working directory and let you view it from the database selector.

    These actions cannot be undone. If you delete or overwrite your data, it is gone unless you made backups. If you delete your stuff, don't come crying to me.
